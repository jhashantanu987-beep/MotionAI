# My SaaS Dashboard

This project is a modern SaaS dashboard built with React, Tailwind CSS, and Framer Motion. It features a responsive layout with smooth animations and interactions, providing a user-friendly experience.

## Features

- **Collapsible Sidebar**: A dark-themed sidebar that can be toggled open and closed with smooth animations.
- **Header**: Displays the title and key statistics at the top of the dashboard.
- **Main Content Area**: Contains various components like LeadCard, TaskCard, and SummarySection.
- **Floating Panel**: A fixed-position panel on the right side for additional content, such as images or videos.
- **Animations**: Utilizes Framer Motion for page load animations, card appearances, hover effects, and more.

## Tech Stack

- **React**: A JavaScript library for building user interfaces.
- **Tailwind CSS**: A utility-first CSS framework for styling.
- **Framer Motion**: A library for creating animations in React applications.

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/my-saas-dashboard.git
   ```

2. Navigate to the project directory:
   ```
   cd my-saas-dashboard
   ```

3. Install the dependencies:
   ```
   npm install
   ```

4. Start the development server:
   ```
   npm start
   ```

5. Open your browser and go to `http://localhost:3000` to view the dashboard.

## Usage

- The sidebar can be toggled by clicking the menu icon.
- Hover over cards to see animations and effects.
- Click buttons for feedback animations.
